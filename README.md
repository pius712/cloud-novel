# 작가 클럽

한 작가가 여러 클럽에

- 생성 가능
- 소속 가능

## 클럽 관리

actor = 관리자

- 생성, open, 가입 승인 등

### 생성

모집

- INIT: 준비단계
    - 가입 설문 필요
- OPEN : 모집중인 상태
- CLOSE : 폐쇄된 상태
- HIDE : 클럽이 가려진 상태

- 미리 지원자의 스펙에 제한을 할 수 있어야함.
- 질문지를 만들어야 함.

가시성

- public 클럽
    - 참가 신청 시, 가입
- private 클럽
    - 참가 신청 시, 승인에 의해 가입

## 클럽에서의 활동

actor = member

가입신청

1. 질문지에 응답 필요
2. private, public 에 따라 즉시가입, 승인 필요함.
